## Uso

Compila y ejecuta el programa:

### Compilación manual

#### Windows

```sh
cd src
gcc p3.c -o p3.exe
p3.exe
```

#### Linux

```sh
cd src
gcc p3.c -o p3.out -lm
./p3.out
```
